JS里面定义的基础类型

|类型|原始类型|引用类型|构造函数|
|:-:|:-:|:-:|:-:|
|BigInt|是|-|-|
|Boolean|是|-|-|
|Null|是|-|-|
|Number|是|-|-|
|String|是|-|-|
|Symbol|是|-|-|
|Undefined|是|-|-|
|Array|-|-|是|
|Date|-|-|是|
|Function|-|-|是|
|Map|-|-|是|
|WeakMap|-|-|是|
|RegExp|-|-|是|
|Set|-|-|是|
|WeakSet|-|-|是|
|String|-|-|是|
|Boolean|-|-|是|
|Number|-|-|是|

```javascript
var { toString } = Object.prototype;
var list = [
  1n,
  false,
  null,
  2,
  '3',
  Symbol('symbol'),
  undefined,
  [],
  new Date(),
  function() {},
  function*() {},
  new Map(),
  new WeakMap(),
  new Set(),
  new WeakSet(),
  new String('s'),
  new Boolean(),
  new Number(2),
];
list.forEach(ele => {
  console.log(toString.call(ele));
})
```
