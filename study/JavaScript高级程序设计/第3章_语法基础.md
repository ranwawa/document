- 区分大小写

### number类型
6种数值字面量
- 0/010 十进制
- 0b1 二进制
- 0o1/01 八进制
- 0x1 十六进制
- 0.1/.1 浮点数
- 2e3/2e+3/2e-3 科学计数

特别注意0开头的整数,07及以下的是8进制 08以上的是十进制

8个数值的边界值
- Number.NEGATIVE_INFINITY
- Number.MIN_VALUE
- -0
- 0
- NaN
- +0
- Number.MAX_VALUE
- Number.POSITIVE_INFINITY

NaN的2个特点
- 和任何数操作结果都是NaN
- 和任何数包括自己都不相等

Number上转换数值的3种方法
- Number构造函数
- Number.parseInt转整
- Number.parseFloat转浮点数

Number构造函数转整数的6个情况
- undefined -> NaN
- null -> 0
- true -> 1, false -> 0
- number直接返回
- 字符串的4个步骤
  - 忽略前面的空格和0
  - 数字转换为10进制,可识别前面的+/-符号
  - 0x/0b也会进行转换得是0o不会,无法识别+/-符号
  - 空字符串转换成0
  - 其它转换成NaN
- 对象先调用valueOf,valueOf不存在再调用toString,然后根据上面的逻辑走一下

parseInt的4个步骤
- 忽略前面的空格
- 如果第一个字符是+/-数字以外的返回NaN
- 碰到非字符或末尾结束
- 可接收第2个参数

parseInt与Number的4个区别

|差异点|parseInt|Number|
|:-:|:-:|:-:|
|+/-号识别|都识别|只有十进制识别|
|空字符串|返回NaN|返回0|
|小数点|返回整数部分|返回浮点数|
|二进制|支持|不支持|

parseInt与parseFloat的4个区别

|差异点|parseInt|parseFloat|
|:-:|:-:|:-:|
|小数点|返回整数部分|返回浮点数|
|科学计数|返回整数部分|返回10进制|
|进制|支持所有进制|只支持十进制|
|底数参数|支持|不支持|

### String类型

3种字符串字面量形式
- 单引号`'`
- 双引号`"`

11种转义字符

- \\" 双引号

+=拼接字符串很慢的3个原因 
- 字符串是不可变的
- 拼接时会创建一个新空间
- 然后再销毁之前的2个变量

2种转换字符串的方式
- toString
- String(本身也是调用toString方法)

String和toString的3个区别
|差异点|toString|String|
|:-:|:-:|:-:|
|undefined|不支持|'undefined'|
|null|不支持|'null'|
|数值|可以带底数参数|没有参数|

乘性操作符的1个类型转换
- 如果不是数值,则调用Number进行转换

加性操作符的2个类型转换
- 如果有一个是字符串,则用String将另外一个转换成字符串
- 如果有一个是数值,另一个不是字符串,则用Number将另一个转换成数值

逻辑操作符的2个比较步骤
- 如果是字符串,则静候对比字符的编码
- 如果任意一个不是数值,则使用Number将其转换成数值,再比较
