# sonar cloud

## 1. 自动检测和基于 ci 检测的区别

|          | 自动检测  | 基于 ci 检测                             |
| -------- | --------- | ---------------------------------------- |
| 支持平台 | 仅 github | 所有平台                                 |
| 配置     | 不需要    | 根目录创建 sonar-project.properties 文件 |

## 2. 不同分支检测的区别

|                  | 主分支 | 长期分支 | pull request | 短期分支 |
| ---------------- | ------ | -------- | ------------ | -------- |
| 全量检测         | o      | o        | x            | x        |
| 增量检测         | o      | o        | o            | o        |
| 是否需要单独配置 | x      | x        | o            | o        |

参考: [官方文档](https://docs.sonarcloud.io/enriching/branch-analysis/)
