# Shell 命令速查

| 命令      | 含义                         |
| --------- | ---------------------------- |
| printenv  | 查看当前系统中的环境变量列表 |
| export    | 设置环境变量                 |
| pwd       | 查看当前目录                 |
| locale    | 查看当前地区设置             |
| locale -a | 查看可以使用的国家/地区列表  |
| declare   | 声明变量                     |
| readonly  | 将变量转换成只读             |

## 复制粘贴相关

| 命令              | 含义                          |
