# Shell 命令速查

| 命令      | 含义                         |
| --------- | ---------------------------- |
| printenv  | 查看当前系统中的环境变量列表 |
| export    | 设置环境变量                 |
| pwd       | 查看当前目录                 |
| ls        | 查看当前目录下的所有文件     |
| ls -l     | 显示详细信息                 |
| locale    | 查看当前地区设置             |
| locale -a | 查看可以使用的国家/地区列表  |
| declare   | 声明变量                     |
| readonly  | 将变量转换成只读             |

## 复制粘贴相关

| 命令              | 含义                          |
| ----------------- | ----------------------------- |
| pbcopy < fileName | `macos`将文件内容复制到剪贴板 |

## 暂未分类

| 命令              | 含义                               |
| ----------------- | ---------------------------------- |
| echo 'xxx'        | 输出内容                           |
| 行尾/             | 用于代码换行                       |
| #                 | 注释                               |
| cd '/'            | 跳转目录                           |
| chmod +x './file' | 为文件添加可执行权限               |
| file './file'     | 判断识别文件类型                   |
| file './file'     | 判断识别文件类型                   |
| echo a \| od      | 以 8 进制方式显示内容              |
| echo a \| od -a   | 以别名显示内容(对不可见字符很有用) |
