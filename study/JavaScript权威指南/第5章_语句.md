### 5.6.6 try catch finally语句

如果finally块使用了return,continue,break,throw语句,服务器会忽略之前的跳转
```javascript

	try{
		throw ...;
	}finally{
		return 1;
	}
}
foo(); //并不会捕获异常,而是返回1
```
